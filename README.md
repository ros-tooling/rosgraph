# ROS 2 Graph Tooling (Declaration, Introspection, Observation) meta-repository

This repository is the entrypoint for design, discussion, and cross-tool requirements management for the ROS Graph Project.

Topics managed here:
- Declarative interfaces for generating nodes
- Data structures for describing instantiated nodes and graphs
- Analysis and linting of ROS graphs
- Introspection and description of launch files/systems
- Health monitoring of ROS systems at the application level
- Documentation generation for nodes and applications
